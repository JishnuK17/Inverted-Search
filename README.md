# Inverted-Search
Inverted Search is a program designed to create a searchable database of words from multiple text files. It supports displaying the database, updating it with new files, searching the database, and saving the database for later use. 
# Features
- Creation of database
- updating the Database with new files
- Searching the databse
- Save the database
# Technologies used
- Linked list
- Hashing
- Dynamic memory allocation
- File handling
